## Vision: Technical Artist Tutor Framework

This plugin is a **proof of concept** for a larger vision:

### Near-term
- Complete coverage of Designer First Steps course
- Robust tutoring for fabric material creation
- Pattern for documenting video tutorial knowledge

### Medium-term
- Generalize process for any video tutorial series
- Template system for creating tutor skills
- Multiple Substance Designer courses

### Long-term Dream
- Technical Artist tutor skills for different 3D topics
  - Maya workflows
  - Houdini procedural techniques
  - Blender materials
- Integration with software SDKs for direct tool execution
- Real-time troubleshooting by viewing user's software screen
